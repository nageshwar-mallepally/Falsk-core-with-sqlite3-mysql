# Falsk-core-with-sqlite3-mysql

### To install Flask

pip install Flask

### To run flask application

flask run


### To run on debug mode and hot swap

export FLASK_DEBUG=1
flask run

### To install spqlite3
sudo apt install sqlite3

### To create DB
sqlite3 DatabaseName.db

### To see databases
.databases

### To my sql connector

pip install mysql-connector-python
